# Prediction of Forest-Related Carbon Emissions Using Machine Learning

## 📌 Project Overview

This project focuses on predicting forest-related carbon emissions using environmental and geographical data through Machine Learning techniques.

The project includes data preprocessing, exploratory data analysis, feature engineering, model development, evaluation, and prediction to identify the most suitable regression model for estimating carbon emissions.

---

## 🎯 Objectives

- Analyze environmental datasets.
- Prepare data for Machine Learning.
- Perform Exploratory Data Analysis.
- Train multiple regression models.
- Compare model performance.
- Predict forest-related carbon emissions.

---

## 📊 Dataset

The dataset contains environmental variables affecting carbon emissions from forest ecosystems.

## 📈 Model Evaluation

Model performance was evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

Among all the implemented models, the **XGBoost Regressor** achieved the highest prediction accuracy.
